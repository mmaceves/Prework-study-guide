HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are different heading elements, each one is used to specify the level.
  • The title goes inside the head element.

CSS

  • The * selector declares that ALL of the elements on the page will have the style properties.
  • The , combines two selectors into a single CSS rule.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The .css file is added in the head element with the link atribute.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: makes sure that our current branch is up to date.
  • git checkout main: returns to the main branch.
  • git add -A: commit all files in the working branch.
  • git commit -m: adds a message with our commit.

JavaScript

  • The .js file is added in the body element.
  • The control flow is from top to bottom in javascipt.
  • Conditional statements an interupt the control flow.
  • An array is a signle varialbe used to hold a group of data.
